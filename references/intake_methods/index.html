Filter data by date¶

IntakeTransform._filter_data_on_datetime¶

Filter data on a given data time value.

Parameters:
  • data_source (string) –

    String that denotes where to pull data from.

  • filter (dict) –

    Dictionary containing the filter information for date restriction. The expected format is a dictionary with keys ['filter_col','operator','filter_date'].

  • map_date_col (str) –

    Name of the column to add the datetime object to.

Examples:

>>> data_source = "Extract.Data Entry"

    Import Data Entry sheets from the extract step.

>>> data_source = "Extract.Lucious6.Data Entry"

    Import extracted data for source Lucious 6 with key 'Data Entry'.

>>> data_source = "Transformations.2"

    Import results from the second transformation step.

>>> filter = {'filter_col':'map_date_col','operator':'==','filter_date':'today'}

    Filter data on map_date_col where it is equal to todays date.

    More generally:

        filter_col (str): Column name to use for filtering. Typically the same as `map_date_col`.

        operator (str): String representation of a python logic operator ('==','<',etc.)

        filter_date (datetime or string*): Date time object or the string 'today', which will infer the current day time object

Aliases: filter_by_date

Update the view of todays intake data¶

IntakeTransform._update_today_view¶

Update todays view of data with mapped start time, date, appointment time, Amount Overtime, and participant type.

Parameters:
  • data_source (str) –

    String that denotes where to pull data from.

  • today_source (str) –

    String that denotes where to pull todays data from.

  • duration_cols (list, default: ['Duration', 'Appt. Duration'] ) –

    List of the actual duration column name, and the scheduled appoint duration column name. Used to get overflow. Defaults to ['Duration','Appt. Duration'].

  • status_col (str, default: 'Status Subtype' ) –

    Column name for the status subtype. Defaults to 'Status Subtype'.

  • mappings (dict, default: None ) –

    Dictionary with method names as keys, pointing to another dictionary. This inner dictionary must have an 'outcol' key, which states the output column name to place the results into. Any additional elements in the inner dictionary get converted into inputs for the requested method. If only 'outcol' is provided, sends the entire row to the method. Defaults to None.

Examples:

>>> data_source||today_source = "Extract.Data Entry"

    Import Data Entry sheets from the extract step.

>>> data_source||today_source = "Extract.Lucious6.Data Entry"

    Import extracted data for source Lucious 6 with key 'Data Entry'.

>>> data_source||today_source = "Transformations.2"

    Import results from the second transformation step.

>>> mappings = {'common_methods.compute_bmi': {'outcol': 'BMI','height_col': "Height (in)",'weight_col': "Weight (lbs)"}}

    Call the method common_methods.compute_bmi, pass the input arguments 'height_col' the column labeled 'Height (in)'.
    Pass the input argument 'weight_col' the column name 'Weight (lbs)', and save results to 'BMI'.

Aliases: prepare_today_data

    String that denotes where to pull data from.

    Column to look for source spreadsheet information. Loops over these sources to send to the right spreadsheet. Defaults to 'source'.
