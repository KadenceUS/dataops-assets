Intake Methods¶

Filter data by date¶

Filter data on a given data time value.

Parameters:
  • data_source (string) –

    String that denotes where to pull data from. Extract.Data Entry would look for any Data Entry sheets made in the extract step. 'Transformations.2` would mean to take the results from the second transformation step.

  • filter (dict) –

    Dictionary containing the filter information for date restriction. The expected format is: filter_col (str): Column name to use for filtering. Typically the same as map_date_col. operator (str): String representation of a python logic operator ('==','<',etc.) filter_date (datetime or string*): Date time object or the string 'today', which will infer the current day time object

  • map_date_col (str) –

    Name of the column to add the datetime object to.

 
Aliases: filter_by_date
