Commands¶

usage: main.py [-h] [--project PROJECT] [--endpoints ENDPOINTS]
               [--env_path ENV_PATH] [--secret_name SECRET_NAME]
               [--extract_google | --extract_aws] [--schedule_summary]
               [--quotas] [--calendar]

options:
  -h, --help            show this help message and exit
  --extract_google      Select google as the data repository
  --extract_aws         Select aws as the data repository

Configuration group:
  --project PROJECT
  --endpoints ENDPOINTS
  --env_path ENV_PATH
  --secret_name SECRET_NAME

  --schedule_summary    Run the schedule summary actions.
  --quotas              Run the intake actions.
  --calendar            Run the calendar actions.
